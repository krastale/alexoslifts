# AlexosLifts

A modern fitness tracking application.

## Features

- **Routine Builder:** Create and manage your workout routines with exercise categories.
- **Workout Logger:** Track your sets, reps, and weight with real-time PR detection.
- **Progress Tracking:** Monitor your strength gains and body measurements.
- **Social:** Connect with friends and share your progress.
- **Minigames:** Stay active during rest periods with fun minigames.

## Tech Stack

- React
- Vite
- Tailwind CSS
- Supabase
- Lucide React
- Recharts

## Getting Started

1. Clone the repository.
2. Install dependencies: `npm install`.
3. Set up your Supabase environment variables.
4. Run the development server: `npm run dev`.
5. Build for production: `npm run build`.
